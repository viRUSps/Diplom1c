# Diplom1c
Диплом

ИНСТРУКЦИЯ ПО НАСТРОЙКЕ 
Все объекты находятся в разделе "Добавленые объекты"

1. Создайте группу и бота в Телеграмм. Они нужны для отправки сообщений специалистам выполняющим работы о созданых и измененных документах "Обслуживание клиентов".
Уведомления создаются при создании нового документа а так же при изменениях даты и ФИО специалиста.

2. Нужно заполнить константы в "Добавленые объекты"-"Сервис"- "Номенклатура обонентская плата" и "Номенклатура работы специалиста"соответствующим наименованием номенклатуры или создать нужную номенклатуру. "Идентификатор группы ТГ" заполняется идентификатором группы телеграмм а "Токен ТБ" заполняется токеном бота телеграмм.

3. Заполните регистр сведений "Условия оплаты сотрудников" укажите оклады всех работиков и процент от работ у Специалистов.

4. В "График работ Справочник" заведите необходимый вид графика работы
