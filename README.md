# Diplom1c
Диплом

ИНСТРУКЦИЯ ПО НАСТРОЙКЕ 
Все объекты находятся в разделе "Добавленые объекты"

1. Создайте группу и бота в Телеграмм. Они нужны для отправки сообщений специалистам выполняющим работы о созданых и измененных документах "Обслуживание клиентов".
Уведомления создаются при создании нового документа а так же при изменениях даты и ФИО специалиста.

2. Нужно заполнить константы в "Добавленые объекты"-"Сервис"- "Номенклатура обонентская плата" и "Номенклатура работы специалиста"соответствующим наименованием номенклатуры или создать нужную номенклатуру. "Идентификатор группы ТГ" заполняется идентификатором группы телеграмм а "Токен ТБ" заполняется токеном бота телеграмм.

3. Заполните регистр сведений "Условия оплаты сотрудников" укажите оклады всех работиков и процент от работ у Специалистов.

4. В "График работ (Справочник)" заведите необходимый вид графика работы который действует у Вас. Полсе этого в подсистеме Сервис выберите "Заполнение графика". В полях выбора "Выбор периода" выберите необходимый Вам период для заполнения графика. В поле "Выходные дни" через запятую перечислите порядковый номер дня недели для выходного (6,7 суббота и воскресенье). В поле "График работы" выберите ранее созданый или создайте жлемент справочника "Графики работы (Справочник)" и нажмите кнопку заполнить. За тем в подсистеме "Расчеты" в регисте "Графики работы" проверьте корректность заполнения созданого Вами графика. Для расчетов по другим графикам необходимо отдельно создать и заполнить необходимые вам графики.

5. В "Администрирование" - "Настройка пользователей и прав" - "Пользователи" создайте необходимых вам пользователей "Полное имя" полное ФИО пользователя "Имя (для входа)" короткое под которым пользователь будет заходить в базу ("Полное имя" Иванов Иван Иванович "Имя (для входа)" Иванов И.И.). В Права доступа установите права доступа пользователя Профили созданы интуитивно понятными (Администратор, специалисты, Менеджер по продажам).

6. Сообщите бухгалтеру о возможности автоматического создания актов
